# 회원관리 프로그램
#사용언어
  javafx + scenebuilder + Oracle
#프로그램 설명 
  회원관리를 위한 프로그램 구현
  
  [프로세스]
  
  (1) SceneBuilder 프로그램을 이용하여 fxml을 만든다.
  
  (2) Eclipse에서 main 자바 파일에서 FXML를 활용하여 실행한다.
  
  (3) 실행이 완료될 경우 Controller파일을 생성한다.
